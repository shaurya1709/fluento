:root {
	--main-color: /* the main color. used for things like backgrounds */
	--accent-color: /* the accent color. used for things like navbar backgrounds */
	--text-color: /* the color of the font used across the webpage */
	--text-color-alternate: /* an alternate font color used for navbar hovers, active navbar urls and button text */
	--font: /* the font family to be used across the webpage */
	--page-margin: /* the margin used by the page body */
	--text-case-filter: /* text casing filters. possible values are capitalize, lowercase, uppercase and none */
	--scroll-style: /* set smooth scrolling for anchors leading to other elements on the page. possible values are smooth, auto */
}

the active iD is used inside a navbar to specify the active page and its styling.

class pairs

class pairs combine two or more classes to style the element differently.

glow

adds a glow effect to the element on hover

Applicable Elements

button, navbar

code

<ul class="navbar glow">
	<li><a href="page 1 url">Page 1</a></li>
	<li><a id="active">Page 2</a></li>
</ul>

Combining Class Pairs

class pairs can be combined as well. example -
a navbar with classes sticky and glow is possible! even the navigation bar on this webpage combines class pairs.

how to combine class pairs

an example to do so is a navbar. navbars are made when the class navbar is added to a list.

example

navbar + sticky = sticky navbar.

navbar + glow = navbar with glow in hovered links.

therefore, navbar + sticky + glow = the properties of sticky and the properties of glow added to the navbar

example

animations

Getting Started

to use animations, just add the animation class along with the class for the animation required. animations are also class pairs. an animation requires four classes. they are animation, animation length, animation type and animation iteration count

Animation Length

possible values: regular, slow, slower, fast, faster. can be also set using the animation-duration property

Animation Type

possible values: spin, fadeback.

spin

Class: spin

spins the element infinitely

fadeback

Class: fadeback

fades the text in and out.

Animation Iteration Count

possible values: once, infinite. can also be set using the animation-iteration-count property.
